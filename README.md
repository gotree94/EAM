# Enterprise Asset Management, 전사적 자산 관리
---

## 1. e-Maintenance (e-메인터넌스)
 - 가장 직접적으로 'e'로 시작하는 용어입니다.
 - 인터넷 및 무선 기술(ICT)을 활용하여 원격으로 설비 상태를 모니터링하고 유지보수하는 포괄적인 체계를 의미합니다. 

## 2. EAM (Enterprise Asset Management, 전사적 자산 관리)
 - 예지보전을 포함하여 기업의 모든 설비 자산을 생애주기별로 통합 관리하는 시스템입니다.
 - 단순히 고장을 고치는 것을 넘어, 자산의 가치를 최적화하는 데 초점을 맞춥니다. 

## 3. Early Warning System (조기 경보 시스템)
 - 예지보전의 핵심 기능 중 하나로, 설비의 미세한 이상 징후를 감지하여 고장이 나기 전 미리 경고(Early Warning)를 보내주는 시스템을 뜻합니다. 

## 추가로 확인해 볼 용어들:
 - Edge Computing (엣지 컴퓨팅): 설비 현장에서 데이터를 즉시 분석하여 지연 시간 없이 예지보전을 수행하는 기술입니다.
 - Emergency Maintenance (긴급 보전): 예지보전에 실패했을 때 발생하는 돌발 고장 수리를 의미하며, 보통 예지보전 자료에서 비교 대상으로 자주 등장합니다.
